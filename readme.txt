Weather aggregator shows parsed weather information from three different weather
providers.

Providers: Accuweather, RP5 and Sinoptik.
You can set any of them by default.

Weather information:
  Current weather: temperature, real feel and condition;
  Forecast for next 8 hours (24 - RP5): min, max and average temperature;
  Next day forecast: min, max and condition (also real feel by Accuweather).

Language: Ukrainian.

Usage.
You have to install Python 3.6. or later.
Type 'python app.py -h' for help.
